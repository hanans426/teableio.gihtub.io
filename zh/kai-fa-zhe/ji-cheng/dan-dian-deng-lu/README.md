# 单点登录

目前Teable提供三种单点登录配置方式，分别适用于使用不同的Teable版本。

| 配置方式    | 适用对象         | 配置入口   | 特点         |
| ------- | ------------ | ------ | ---------- |
| 环境变量配置  | 所有自部署用户      | 部署配置文件 | 全局生效，统一管理  |
| 管理员后台配置 | 自部署专业版及以上用户  | 管理员后台  | 灵活配置，可动态修改 |
| 空间站配置   | 云托管专业版及以上空间站 | 空间站设置  | 按空间独立配置    |
